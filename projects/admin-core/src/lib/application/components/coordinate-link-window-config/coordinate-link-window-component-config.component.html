Add one or more URL's to configure the Coordinate Link Window component.
For each URL you can use [X] or [lat] and [Y] or [lon] placeholders in the URL.
Those will be replaced by the location on which the user clicks.
For example: https://www.google.com/maps/@[lat],[lon],18z or https://demo.tailormap.com/#@[X],[Y],18
You can also use [GPS-X], [GPS-Y], [GPS-lat] and [GPS-lon] placeholders in the URL.
Those will be replaced by the current device location.
[GPS-X] and [GPS-Y] use the configured projection, [GPS-lat] and [GPS-lon] use WGS84.
For example, to open a route from your device location to the clicked location in Google Maps: https://www.google.com/maps/dir/[GPS-lat],[GPS-lon]/[lat],[lon]
